# Airport Management Information System

This repository contain SQLite database project for Airport Management Information System.

## Files

- airport_management.sql: contain DDL DML and management queries.
- Project_Report.docx: short project report.
- airport_erd.dbml: ERD code for dbdiagram.io.

## How to run

Open SQLite and run airport_management.sql.

## Main SQL topics used

SELECT WHERE ORDER BY LIKE BETWEEN JOIN GROUP BY HAVING Subquery UNION and SQL functions.
